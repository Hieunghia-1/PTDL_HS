1. Công nghệ sử dụng

Ngôn ngữ: Python

Môi trường: Jupyter Notebook

Thư viện chính:

pandas: đọc và xử lý dữ liệu

numpy: hỗ trợ tính toán

2. Cách hoạt động

Dữ liệu được đọc từ file CSV vào DataFrame bằng thư viện Pandas.

Tiến hành kiểm tra dữ liệu (số dòng, số cột, giá trị thiếu).

Thực hiện các bước xử lý và phân tích dữ liệu theo yêu cầu bài lab:

Sắp xếp dữ liệu

Thống kê các giá trị như: count, mean, min, max, median, quartile

Dữ liệu sau khi xử lý được trình bày bằng bảng và biểu đồ để dễ quan sát.

3. Kết quả

Dữ liệu được làm sạch và sắp xếp đúng yêu cầu.

Các giá trị thống kê mô tả phản ánh rõ đặc điểm của tập dữ liệu.

Biểu đồ giúp quan sát phân phối và xu hướng dữ liệu một cách trực quan.

Kết quả đáp ứng đầy đủ mục tiêu của bài thực hành.
