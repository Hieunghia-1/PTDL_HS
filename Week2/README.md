# Lab 2 – Phân tích và trực quan hóa dữ liệu tuyển sinh đại học


## Công nghệ sử dụng
- Python
- Jupyter Notebook (VS Code)
- Thư viện: pandas, matplotlib, scipy

## Cách hoạt động
- Trực quan hóa dữ liệu theo nhóm (giới tính, khu vực, khối thi, dân tộc).
- Mô tả thống kê và khảo sát phân phối biến T1:
  - Các đại lượng thống kê mô tả.
  - Box-plot, Histogram, QQ-plot.
- Phân loại điểm T1 và phân tích theo từng nhóm phân loại.
- Khảo sát tương quan giữa T1 và DH1:
  - Hệ số tương quan.
  - Biểu đồ Scatter.
- Khảo sát tương quan giữa DH1, DH2, DH3.

## Kết quả
- Dữ liệu được mô tả và trực quan hóa rõ ràng.
- Biến T1 có phân phối không hoàn toàn chuẩn và có xu hướng lệch nhẹ.
- Tồn tại mối tương quan tuyến tính dương giữa T1 và DH1.
- Các biến DH1, DH2, DH3 có mối liên hệ nhất định với nhau.
